Tive que usar o Transaction pooler do supabase devido aos bloqueios de ipv6 e ipv4 padrão da minha máquina,
e o pooler é a única maneira de contornar isso.
O pooler é um serviço que gerencia conexões de banco de dados e pode ser configurado para aceitar conexões IPv4,
mesmo que o banco de dados esteja configurado para IPv6.
